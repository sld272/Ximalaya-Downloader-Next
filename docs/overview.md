# 喜马拉雅音频下载器 · 项目概览

> Ximalaya-Downloader-Next　|　状态：开发中
> 一个稳定、可扩展、便于长期维护的喜马拉雅音频下载工具。

## 简介

喜马拉雅官方没有提供将音频保存到本地的功能。本项目提供一个命令行工具与可复用的 Python 库，帮助你把**自己有权访问的**音频（免费内容、已购或会员可听内容）下载并整理到本地，支持单曲与整张专辑、断点续传、专辑增量更新等。

整体采用「核心库 + 薄前端」结构：能力沉淀在独立、可测试、可扩展的核心库中，命令行只是其上的一层薄壳，未来的图形界面、Web 或 HTTP API 都可复用同一套核心。项目背景与愿景见 [`README.md`](../README.md)，架构与设计原则见 [`architecture.md`](./architecture.md)。

## 核心特性

- **单曲与专辑下载**：按链接或 ID 下载单个音频，或整张专辑、专辑中的指定区间。
- **音质选择**：在可用音质之间选择，并在目标音质缺失时自动回退。
- **断点续传**：任务级、字节级、专辑级三层续传，中断后可从断点继续。
- **专辑增量更新**：专辑有新内容时再次运行只补下新增部分，不重复下载。
- **可靠的下载管理**：下载状态持久化到本地数据库，失败按类型分级重试。
- **克制的请求策略**：可配置的并发、速率控制与退避，默认值偏保守。
- **登录与凭据管理**：支持多种登录方式并安全持久化会话，用于访问会员/已购内容。
- **库 + CLI 双形态**：既是命令行工具，也是可在脚本中导入的 Python 库。

## 快速开始

> 项目开发中，以下为目标使用方式。

安装（开发者）：

```bash
pip install -e .
```

命令行用法：

```bash
xdl login                              # 登录并保存会话
xdl track <链接或ID>                    # 下载单个音频
xdl album <链接或ID>                    # 下载整张专辑
xdl album <链接或ID> --range 1-20       # 下载专辑第 1–20 个音频
xdl album <链接或ID> --quality high     # 指定音质
xdl search <关键词>                     # 搜索
xdl resume                             # 继续上次未完成的下载
xdl config                             # 查看/修改配置
```

作为库使用：

```python
from xdl import Facade

app = Facade.from_config()
await app.download_track(track_id, quality="standard")
await app.sync_album(album_id, quality="standard")
```

| M6 | 本地文件解码备用通道 | ☐ 待做 |

> **当前 MVP 实现**：CLI 提供 `xdl login`、`xdl track`（单曲）与 `xdl album`（专辑顺序批量下载，
> 支持 `--range` 区间与 `--quality` 音质，已存在文件自动跳过 = 文件级续传）。
> `search`/`resume`/`config` 等命令为路线图目标，尚未实现；专辑下载尚无并发/限速/字节级续传/增量同步。
> 早期验证用的单文件原型 `download.py` 仍保留可用，逻辑已迁入 `src/xdl/`。
