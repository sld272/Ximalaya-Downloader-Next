## 简介

喜马拉雅没有官方的下载功能。这个项目可以把你有权限听的内容（免费、已购、会员可听）下载到本地。

[Ximalaya-Downloader](https://github.com/sld272/Ximalaya-Downloader) 是我高中时写的不成熟项目，后来由于无暇维护，就把仓库归档了。现在将项目重启，命名为 Ximalaya-Downloader-Next，完全重写。

## 目标

旧版最大的毛病是结构撑不住平台变化，这次重启主要想解决这个问题，并增加可靠性。

## 项目状态

非常早期。目前确定了基本架构，预计一到两个月内完成基本功能，完整路线图见 [overview.md](./docs/overview.md)。
